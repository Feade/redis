## 安装编译工具
```
  安装 blade
  git clone https://github.com/chen3feng/typhoon-blade.git
  cd typhoon-blade && ./install

  安装scons
  wget http://prdownloads.sourceforge.net/scons/scons-2.2.0.tar.gz
  tar zxvf scons-2.2.0.tar.gz
  cd scons-2.2.0
  python setup.py install
```
## 编译
```
  blade build ...
```
